# MiTienda - Amazon 2024 (Demo)

Proyecto demo de tienda estilo Amazon (visualmente similar pero 100% original).

## Contenidos
- index.html, producto.html, carrito.html, checkout.html, admin_products.html  
- style.css, main.js, productos.js, admin_products.js

## Cómo usar localmente
1. Crea una carpeta y pega los archivos.
2. Abre `index.html` en tu navegador.
3. El carrito y wishlist se guardan en localStorage.

## Cómo publicar en GitHub Pages
1. Crear repositorio público.
2. `git init` en tu carpeta, `git add .`, `git commit -m "init"`.
3. `git remote add origin https://github.com/TU_USUARIO/NOMBRE_REPO.git`
4. `git push -u origin main`
5. En GitHub → Settings → Pages: Branch: main / Folder: root.
6. Accede a `https://TU_USUARIO.github.io/NOMBRE_REPO/`
